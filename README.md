# TripOS

Dynamic Travel Planning & Experience Engine.

## Tech Stack
- Frontend: React + Vite + Tailwind CSS
- Backend: Node.js + Express.js

## Local Setup
1. Run `npm install` in the root directory to install `concurrently`.
2. Run `npm install` in the `client` directory.
3. Run `npm install` in the `server` directory.

## Scripts
- `npm run dev`: Runs both client and server concurrently in development mode.
- `npm start`: Starts the backend server.
- `npm run build`: Placeholder for future build process.
